Application Overview
====================
Perfy is a Windows/WPF editor for designing circuits on Perf+ prototype board (invented and developed by Ben Wang). The latest version of the code and binaries are available from the following GitHub repository: 

https://github.com/Myndale/Perfy

For information on how to use Perfy please refer to the in-app help documentation. All questions/bug reports etc should be directed to perfy@ppl-pilot.com.

For information on Perf+ please refer to Ben Wang's Kickstarter page at https://www.kickstarter.com/projects/658903329/perf-the-perfboard-reinvented/description
